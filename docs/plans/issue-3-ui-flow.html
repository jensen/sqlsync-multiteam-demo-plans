UI Flow: Labels Feature
1
User navigates to Team → Labels
→ LabelManager screen renders
2
User creates a label (name + color)
→ AddLabel mutation sent
3
User creates an issue
→ LabelPicker shows available labels
4
User selects labels in picker
→ Issue saved; AddLabelToIssue mutations
5
Issue list displays LabelChip on each row
→ Filter bar shows active label filters
6
User clicks label filter
→ Issue list re-queries with label JOIN + WHERE
